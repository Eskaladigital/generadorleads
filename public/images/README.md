# Imágenes del sitio

## Hero de la home

Por defecto se usa una imagen de Unsplash. Para usar tu propia imagen:

1. Añade `hero-spain.jpg` (o `AdobeStock_85606851.jpeg`) en esta carpeta
2. En `src/lib/constants.ts` cambia `HERO_IMAGE_URL` a `'/images/hero-spain.jpg'`

Formato recomendado: 1920×1080 px, costa española o Mediterráneo.
